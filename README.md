# Personal Page Template

This repository now contains a GitHub Pages-ready personal homepage inspired by the structure of `https://onground-korea.github.io/`.

## Files

- `index.html`: page structure
- `styles.css`: layout and visual styling
- `site-data.js`: all editable content
- `main.js`: renders the content into the page

## Quick customization

1. Open `site-data.js`.
2. Replace the placeholder name, bio, email, links, education, experience, publications, and projects.
3. If you have a profile photo, set `profile.photo` to an image URL.
4. Push to GitHub Pages.

## Why this setup

The repository was empty, so a plain HTML/CSS/JS site is the fastest path to a deployable page without adding a Jekyll or Node build step.
